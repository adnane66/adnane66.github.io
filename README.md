# Portfolio Personnel - ELOUALI ADNANE 

Bienvenue sur le dépôt de mon portfolio personnel ! Ce site web interactif a pour but de présenter mon profil, mon parcours académique, mes compétences techniques, ainsi que mes projets développés dans le cadre de mes études.

 **Voir le site en direct :** [Mon Portfolio](https://adnane66.github.io/)



##  À propos du projet

Ce portfolio a été conçu pour être à la fois moderne, réactif (adapté aux mobiles) et performant. Il intègre des composants dynamiques gérés par React.js directement dans le navigateur, démontrant ma capacité à mélanger des technologies web classiques avec des bibliothèques modernes.

### Fonctionnalités principales :
* **Navigation fluide :** Menu interactif avec défilement doux (smooth scrolling) vers les différentes sections.
* **Composants React :** Utilisation de React pour le rendu dynamique de la liste des projets, de la section expériences et du formulaire de contact.
* **Design Responsive :** Interface qui s'adapte parfaitement aux ordinateurs, tablettes et smartphones.
* **Barres de progression animées :** Affichage visuel du niveau de maîtrise des différentes compétences techniques (C/C++, Python, HTML/CSS, JavaScript).



## Technologies utilisées

* **Langages de base :** HTML5, CSS3, JavaScript (ES6+)
* **Bibliothèques & Frameworks :** * React 18 (via CDN)
  * ReactDOM (via CDN)
  * jQuery (pour la manipulation simplifiée du DOM)
* **Outils complémentaires :** * Babel (Transpilateur pour JSX)
  * FontAwesome (Icônes)
  * Google Fonts (Police 'Poppins')
* **Hébergement :** GitHub Pages
